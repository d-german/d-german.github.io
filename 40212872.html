Hyland Clinician Window : Using SelectMany()

Created by Damon German, last modified on Feb 23, 2022


Here we are combining several lists into a single list and returning it. rule(config) returns IEnumerable<ValidationError>.

This code is very imperative. We are creating and mutating state.


Whenever you're dealing with multiple lists that need to be flattened into a single list, consider using SelectMany().


The SelectMany() method is used to "flatten" a sequence in which each element of the sequence is an underling sequence, like a two-dimensional array or a list of lists.


Of course, if you need to transform each element before flattening, you can use a normal Select().
